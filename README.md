# nettlib: 基于muduo库实现C++11版本的轻量网络库
> **version：0.1**
>
> Google风格编码

## 更改方面：
1. 时间戳实现封装C++11中的\<chrono>
2. 原子操作即采用C++11中的atomic对int和long类型的特化版本
3. 使用C++11的mutex以及condition替代muduo封装的对应的类型



## git 提交规范：
- 特性新增: 增加了新的功能
- 问题修复: 修复了bug
- 文档补充: 文档补充
- 格式调整: 格式（不影响代码运行的变动）
- 优化重构: 重构（不是新增功能，也不是修复bug的代码变动）
- 测试新增: 增加了新的测试
- 辅助变动: 构建过程或辅助工具的变动
